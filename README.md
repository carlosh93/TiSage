# TiSage

Code and reproducibility material for the paper Semi-Supervised Tissue Segmentation with Multi-Scale Semantic Guidance (Paper ID: 3280), submitted to MICCAI 2026

## Structure

- `LUTSeg/`: dataset reproducibility from raw Label Studio exports, including expert voting workflow for golden-set annotation selection.
- `method/`: proposed TiSage method code and experiments (to be added).

## Quickstart

- Install dependencies:
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ```
- Dataset construction details + examples: `LUTSeg/README.md`
- Inter-rater agreement figure: `LUTSeg/inter_rater_figure1.png` (generated by `LUTSeg/pipeline/luts_inter_rater_figure1.py`)

## Data Availability

- The full dataset is not included in this repository during double-blind review.
- A small, non-identifying subset is provided under `LUTSeg/examples/` to document file formats and the voting workflow.
