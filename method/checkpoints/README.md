# Checkpoints

Default MedSigLIP classifier checkpoints used by training and evaluation scripts are:

- `pretrained/medsiglip_head_dfutissue.pt`
- `pretrained/medsiglip_head_lutseg.pt`

The full expected paths from repository root are:

- `method/checkpoints/pretrained/medsiglip_head_dfutissue.pt`
- `method/checkpoints/pretrained/medsiglip_head_lutseg.pt`

You can override these paths with CLI arguments.
