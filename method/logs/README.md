# Logs

This folder stores logs used to support reported TiSage results.

## Main Table Logs

- `main_table/`: evidence logs linked from `method/README.md` for each value in the main comparison table.
- Files are organized as:
  - `main_table/<method>/<dataset>/<split>/out.log`

## Anonymization

- Logs in this folder are sanitized for double-blind review.
- Reported metrics are preserved.
